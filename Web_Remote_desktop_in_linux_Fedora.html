Remote desktop in linux Fedora

Всем привет! В данном посте рассматривается ПО для подключения к удалённым рабочим столам.

Нижеописанное ПО рассматривалось на OC Linux Fedora 36 Workstation

Все описанные ниже действия выполнялись под пользователем root

Решил написать пост о том, что на Linux Fedora Worktation 36 можно установить пакет rdesktop, с помощью которого можно подключиться по протоколу rdp к Windows Server.

Конечно, в Linux Fedora есть и другие клиенты для соединения с удалёнными рабочими столами: Remmina, x11vnc, TigerVNC... Данный список можно долго перечислять, но далеко не все могут без проблем организовать удаленное соединение без ошибок и с качественной картинкой. Могу сказать лишь от себя, что rdesktop подходит тем, кто любит повводить команды в консоли) А теперь плавно перейдем к установке.
# yum install -y rdesktop Команда установки пакета rdesktop

Ну и самое интересное - это команда подключения к серверу)
# rdesktop -z -P -g 1280x900 -u USERNAME -p PASSWORD SERVER_IP За что какие ключи отвечают?
1) -z включает сжатие потока;
2) -P кэширование;
3) -g задает геометрию окна;
4) -u имя пользователя на сервере;
5) -p пароль.
# rdesktop -r disk:share=/home/documents -z -P -g 1280x900 -u USERNAME -p PASSWORD SERVER_IP При помощи ключа "-r disk:sharename=path" можно подключить любую локальную папку как и диск Windows, который можно будет увидеть в "Мой компьютер".

Если к удаленному рабочему столу подключение происходит давольно часто, то можно автоматизировать данное соединение, написав простой скрипт для подключения)
#!/bin/bash
rdesktop -z -P -g 1280x900 -u USERNAME -p PASSWORD SERVER_IP

Ну и после его написания, не забудьте сделать его исполняемым.
# chmod 777 remote.sh Вместо "remote.sh" прописываете название своего текстового файла с кодом.

Thank you!)
