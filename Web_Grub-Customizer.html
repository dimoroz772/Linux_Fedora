Grub-Customizer

Всем привет! В данном посте рассматривается установка и настройка пакета "Grub-Customizer" на OC Linux Fedora Workstation 38.

Заходим в командную строку:
# su Выполните команду su и введите пароль от root пользователя.
# yum install -y grub-customizer Команда для установки пакета Grub-Customizer.
# nano /etc/default/grub Команда для перехода в конфигурационный файл загрузчик операционной системы. Нужно внести правки в конфигурационный файл GRUB-загрузчика для того, чтобы в ПО "grub-customizer" показались все разделы загрузки Linux Fedora и их можно было редактировать.

Далее заменяем в строке GRUB_ENABLE_BLSCFG="true" заменяем значение "true" на "false". Должен получиться такой файл:
GRUB_TIMEOUT="5"
GRUB_DISTRIBUTOR="$(sed 's, release .*$,,g' /etc/system-release)"
GRUB_DEFAULT="saved"
GRUB_DISABLE_SUBMENU="true"
GRUB_TERMINAL_OUTPUT="console"
GRUB_CMDLINE_LINUX="rhgb quiet"
GRUB_DISABLE_RECOVERY="true"
GRUB_ENABLE_BLSCFG="false"

Сохраняем файл ↑ и выходим в командную строку.

Далее вводим в командной строке вводим команду "grub-customizer", ждем открытия ПО и меняем нужные настройки.

Всем спасибо за внимание!
