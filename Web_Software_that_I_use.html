Software that I use

Всем привет! Хочу поделиться информацией о том, какое стороннее ПО я использую на Linux Fedora.

Большинство ПО я искал на замену Windows приложениям.

1. Angry IP Scanner — ПО для сканирования сети

2. CPU-X — ПО для просмотра информации о PC

3. Drawio — ПО для рисовки графиков

4. Firefox — Браузер

5. GNU Image Manipulation Program — Графический редактор

6. GtkStressTesting — ПО для нагрузочного тестирования

7. HPLIP Graphical Tools|hp-uiscan — ПО для работы с принтерами HP

8. Htop — ПО для просмотра запущенных задач на PC

9. Krita — Графический редактор

10. LibreOffice — Офисный пакет

11. MIB Browser — ПО для просмотра иерархии SNMP MIB-переменных в древовидной форме

12. Notepad++ — Текстовый редактор

13. OBS Studio — ПО для записи видео и потокового вещания

14. PlayOnLinux — Надстройка над Wine

15. Putty — ПО клиент для различных протоколов удалённого доступа

16. System Profiler and Benchmark — ПО для просмотра информации о PC

17. Telegram Desktop — Мессенджер

18. Snap Store — Магазин сторонних приложений для Linux

19. VMware Workstation Pro — ПО виртуализации

20. Wine — ПО для установки/запуска Windows-приложений на Linux

21. WinSCP — Свободный графический клиент протоколов SFTP и SCP

22. VLC media player — Медиаплеер

23. FreeRDP|xFreeRDP — Клиент удалённого рабочего стола

24. KRDC — Приложение-клиент для просмотра и управления сеансами на других ПК

25. Typora — Текстовый редактор, который позволяет писать исходным кодом Markdown

26. Heicy — Конвертер фото из формата HEIC в формат PNG

27. Visual Studio Code — Редактор исходного кода

28. Media Controls — Средство для управления медиа на Linux Fedora(под Gnome)

29. Grub-Customizer — Утилита для редактирования загрузчика ОС
