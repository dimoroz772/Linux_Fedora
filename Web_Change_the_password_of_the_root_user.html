Change the password of the root user

Всем привет! В данном посте рассматривается то, как сменить пароль у пользователя root.

Заходим в командную строку и вводим следующую команду:
$ sudo passwd root Команда для смены пароля у пользователя root. После выполнения команды, введите два раза новый пароль от для пользователя root, каждый раз подтверждая действие.
