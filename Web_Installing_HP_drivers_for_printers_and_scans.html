Установка драйверов для принтеров/сканеров HP на ОС Linux Fedora -------------------------------------------------------------------------------------------------------------------------------------------------------------- Все описанные ниже действия выполнялись под пользователем root -------------------------------------------------------------------------------------------------------------------------------------------------------------- yum search hplip |#|#| Команда для поиска пакета "hplip" в подключенных репозиториях ОС ------------ yum install -y hplip hplip-gui |#|#| Команда для установки пакетов "hplip" and "hplip-gui" ------------ hp-setup |#|#| Данная команда запускает ПО. Также можно воспользоваться запуском ПО через графический интерфейс. ------------ hp-setup -b usb |#|#| Воспользуйтесь данной командной для настройки/подключения принтера, если он подключен через USB-интерфейс ------------ hp-setup 10.10.10.10 |#|#| Воспользуйтесь данной командной для настройки/подключения принтера, если он подключен к сети организации ------------ hp-setup hp10.gray.home |#|#| Воспользуйтесь данной командой для настройки/подключения принтера по его FQDN/Host Name -------------------------------------------------------------------------------------------------------------------------------------------------------------- На этом гайд закончен!
