Creating VLAN Interfaces

Всем привет! В данном посте рассматривается то, как настраивается VLAN на сетевом интерфейсе. Изначально смотрим как называется сетевой интерфейс.
# ip a Просмотр всех IP-адресов, связанных с сетевыми интерфейсами.

В моём случае будет использоваться сетевой интерфейс enp1s0.

# nmcli con add type vlan con-name enp1s0.1 ifname VLAN1 id 1 dev enp1s0 ip4 10.10.10.10/24 Данной командой создается VLAN #1, с IP-адресом 10.10.10.10 с 24 маской.

Если VLAN #1 будет использоваться как основной сетевой интерфейс, то нужно отключить проводное соединение, дополнить VLAN-интерфейс и перезагрузить его:

1.В консоли вводим команду nmtui;
2.Изменить подключение;
3.Во вкладке VLAN выбираем enp1s0.1;
4.Далее заполняем строки: Шлюз, Серверы DNS, Домены по необходимости;
5.ОК;
6.Назад;
7.Активировать подключение;
8.Проводное соединение > Отключить;
9.VLAN (VLAN1) > Отключить > Включить;
10.Назад;
11.Выйти > OK.		 Порядок выполнения команд.

Также нужно отключить автоматическое подключение проводного соединения.

# nmcli device status Проверка активности всех сетевых интерфейсов.
